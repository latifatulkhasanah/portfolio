# Portfolio — Latifatul Khasanah

Portfolio pribadi untuk Digital Marketing Specialist / Performance Marketer, dibuat dengan HTML & CSS murni (tanpa framework), di-hosting gratis lewat **GitHub Pages**.

## Struktur file
- `index.html` — seluruh halaman portfolio (HTML + CSS + JS ada di satu file ini)
- `CV-Latifatul-Khasanah.pdf` — file CV yang bisa diunduh dari tombol "Unduh CV"

## Yang wajib diedit sebelum publish
Buka `index.html`, cari teks yang diapit tanda kurung siku `[ ]` dan bergaris bawah putus-putus di bagian "Studi Kasus" — ganti dengan hasil/angka nyata dari campaign kamu (contoh: "+28% efisiensi budget", "+40% brand awareness").

## Cara update konten
Semua teks ada langsung di `index.html` dalam bahasa HTML biasa — cari teksnya lewat Ctrl+F/Cmd+F di GitHub code editor, lalu ganti.

## Live site
Setelah GitHub Pages aktif, portfolio ini bisa diakses di:
`https://<username-github-kamu>.github.io/<nama-repo>/`
